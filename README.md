# Sistema de Notas em Java

Projeto simples em Java que calcula a média de notas de alunos e determina:

- Média
- Situação (Aprovado, Recuperação ou Reprovado)
- Maior nota
- Menor nota

## Tecnologias usadas
- Java
- ArrayList
- Métodos
- Estruturas condicionais

## Objetivo
Projeto feito para praticar lógica de programação e conceitos básicos de Java.
